Problem 1

Analyze the performance for the best, average and worst case scenarios when you delete an element from a growable array. Consider the 2 possibilities below when doing this analysis. 

/** 
  * Removes the element found at the specified index-th position by replacing it with a null value.
  * @param index     The index in the array of the element to be removed.
  * @return          The object that was deleted.  
  */
Object removeAt(int index);

/** 
  * Removes the element found at the specified index-th position by shifting all elements that come after it to the "left". 
  * That is, if we are removing the 2nd element, the 3rd element becomes the 2nd, the 4th element becomes the 3rd and so on. 
  * Finally, the size of the array is decreased, so we know that next time we add, we can overwrite what was the last element 
  * of the array.
  *
  * @param index     The index in the array of the element to be removed.
  * @return          The object that was removed. null if the the specified index does not exist in the array.  
  */
Object removeAt(int index);

Problem 2

Even though the first version of removeAt is more efficient, Java implements the second version. Discuss what the drawbacks of the first version of removeAt are, why it is difficult to work with, and not suitable as being the standard implementation in a language.

Problem 3

The add(Object item) method in an ArrayList needs to keep track of the underlying array and when it gets full, it replaces it with a new, bigger array in which all elements from the old array are copied. On the first day of classes, we discussed two algorithms to determine the size of the new array. In the first, the new array is 1 larger than the old array (an N element array is replaced by an (N+1) element array). In the second algorithm, the new array is twice larger than the old array (an N element array is replaced by an 2N element array). What is the time complexity of add(Object item) when using the first array expansion method (replacing with an N+1 element array)? Describe in detail how you arrived at your answer.

Bonus: Do the same for the second version of the expansion algorithm (replacing the underlying array with an 2N-element array). Which method is better? Since this is a more difficult analysis this bonus question is not required. We will discuss the analysis in the discussion section.
